# Pulso Ecommerce

## Autor
Antonella Bozzato

## Repositorio
pulso-ecommerce

## Proyecto
Pulso Ecommerce es una aplicación web estática desarrollada como primera etapa de un trabajo práctico de la materia Aplicaciones Web.  
El proyecto representa la estructura inicial de una tienda online de ropa streetwear, incluyendo página principal, páginas por categoría, formulario de login, formulario de registro y navegación entre secciones.

## Descripción
La tienda **Pulso** fue pensada con una estética moderna y oscura, orientada al estilo urbano streetwear.  
En esta primera entrega se desarrolló la estructura general del sitio utilizando HTML y CSS.

El sitio cuenta con:

- Página principal o **Home**
- Página de **Login**
- Página de **Registro**
- Página de categoría **Remeras**
- Página de categoría **Buzos**
- Página de categoría **Pantalones**
- Navbar presente en todas las páginas
- Logo e identidad visual de la marca

## Objetivo del trabajo
El objetivo de esta etapa fue construir la base visual y estructural de un ecommerce, organizando las distintas páginas del sitio y aplicando estilos para mejorar la presentación.

## Tecnologías utilizadas
- **HTML5**
- **CSS3**

- Remeras

## Aclaración
Este proyecto corresponde a una primera etapa de desarrollo, por lo que actualmente no incluye conexión a base de datos ni validación real de usuarios.  
Las páginas de login, registro y logout forman parte de la estructura visual y funcional básica solicitada en la consigna.
